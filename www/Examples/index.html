Data sets

The following data sets are ready to be downloaded to your machine and from there be sent to SignS. They were obtained from the appendices of the paper "Predicting Cancer Patient Survival with Gene Expression Data" by Bair and Tibshirani in PLOS, and then reformatted to conform to the requirements of SignS.

Download three example data sets (DLBCL, Breast, AML); this is a 7.1 MB tar.gz compressed file. (In Linux, if using a shell, create a directory for the data and decompress the file there: ej: 

 
mkdir SignsExampleData 
cd SignsExampleData 
mv ../example.data.sets.SignS.tar.gz .  
tar -zxvf example.data.sets.SignS.tar.gz
If you are using some GUI for file management, well, use whatever the GUI says to do. In Windows, WinZIP and WinRAR both understand tar.gz; the free and LGPL'ed 7-zip understads those (and many more) too.
